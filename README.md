# Advanced Nanomaterials Laboratory (Astro)

## Quick start
1) Put images:
- images/hero/1.jpg 2.jpg 3.jpg (optional)
- images/news/001-cover.jpg (optional)

2) Add content:
- content/news/*.md (each file is one page)

3) Run locally:
- npm install
- npm run dev

4) Deploy:
- Push to main -> GitHub Actions deploys to GitHub Pages.
